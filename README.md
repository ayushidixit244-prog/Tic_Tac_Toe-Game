# 🎮 Tic-Tac-Toe Game

A simple, interactive, and user-friendly **Tic-Tac-Toe game** built as a learning project. This implementation allows two players to play the classic game locally.

## ✨ Features
* **Two-Player Mode:** Play locally with a friend turning X and O.
* **Real-time Win Detection:** Automatically detects horizontal, vertical, and diagonal wins.
* **Reset Button:** Quickly clear the board to start a fresh match anytime.
* **Responsive Design:** Works smoothly across mobile, tablet, and desktop screens.

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
